workflowInstanceKey

workflowInstanceKey: string
  • Defined in lib/interfaces.ts:263

Stringified JSON document that will instantiate the variables for the root variable scope of the workflow instance; it must be a JSON object, as variables will be mapped in a key-value fashion. e.g. { "a": 1, "b": 2 } will create two variables, named "a" and "b" respectively, with their associated values. [{ "a": 1, "b": 2 }] would not be a\ valid argument, as the root of the JSON document is an array and not an object.
